* TCCR0 - Timer Counter Control Register
** Counter Select -  set timer prescaler

|------+------+------+---------------|
| CS02 | CS01 | CS00 | Prescaler     |
|------+------+------+---------------|
|    0 |    0 |    0 | Timer stopped |
|    0 |    0 |    1 | /1            |
|    0 |    1 |    0 | /8            |
|    0 |    1 |    1 | /64           |
|    1 |    0 |    0 | /256          |
|    1 |    0 |    1 | /102          |
|------+------+------+---------------|

** Wave Form Generation

|-------+-------+-------------------|
| WGM00 | WGM01 | Mode              |
|-------+-------+-------------------|
|     0 |     0 | Normal            |
|     0 |     1 | PWM Phase Correct |
|     1 |     0 | CTC               |
|     1 |     1 | Fast PWM          |
|-------+-------+-------------------|

TCNT0 - counter
OCR0 - stationary threshold
OC0A - output from compare
